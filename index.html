Intelligent IDS: Automated Log Analysis with n8n and AI

This project is an automated security system designed to analyze network alerts generated by an intrusion detection platform.

  • Problem: Suricata generates a high volume of logs every day, including many false positives, which consumes analyst time.
  • Solution: Suricata is connected to an AI workflow through n8n to automate triage and prioritization.
  • Suricata writes events to a file and n8n automatically detects and reads the new file.
  • n8n sends the data to AI, which analyzes context and identifies whether activity is critical or noise.
  • If a critical threat is detected, the system automatically sends an alert email with a summary and recommended actions.
